In this project, we will demonstrate a pragmatic approach for pairing R with 
big data. We will use R’s familiar dplyr syntax to query big data stored on a 
server based data store, like Amazon Redshift or Google BigQuery.

We will also discuss how to adapt data visualizations, R Markdown reports, 
and Shiny applications to a big data pipeline. We will focus on general 
principles and best practices; we will avoid technical details related to 
specific data store implementations.
